Can get the Date in Javascript.


Der er forskellige måder at få dato og tid på i JavaScript. Her er nogle grundlæggende metoder:

Grundlæggende metoder

new Date()

Opretter et nyt Date-objekt, der repræsenterer den aktuelle dato og tid.

console.log(new Date()); // Viser den aktuelle dato og tid (UTC)

Date()

Returnerer den aktuelle dato og tid som en streng.

console.log(Date()); // Viser den lokale dato og tid

Date.now()

Returnerer antallet af millisekunder.

console.log(Date.now()); // Viser Unix-tid

Eksempler fra kode i klassen

Koden viser, hvordan man kan bruge new Date() til at få specifikke dele af datoen.

Hent nuværende måned (v1)

Her bruges et array af månedsnavne og getMonth() til at finde den aktuelle måned.

Endpoint: /months/v1

const months = ["January", "February", "March", "April", "May", "June", "July", 
    "August", "September", "October", "November", "December"];
app.get("/months/v1", (req, res) => {
  const currentMonth = months[new Date().getMonth()];
  res.send({ data: currentMonth });
});

Hent nuværende måned (v2)

Her bruges toLocaleDateString() til at formatere måneden som en lang streng.

Endpoint: /months/v2

app.get("/months/v2", (req, res) => {
  const currentMonth = new Date().toLocaleDateString("en-uk", { month: "long" });
  res.send({ data: currentMonth });
});

Hent nuværende måned (v3)

Her bruges Date() og split() til at hente månedsnavnet fra en streng.

Endpoint: /months/v3

app.get("/months/v3", (req, res) => {
  const currentMonth = Date().split(" ")[1];
  res.send({ data: currentMonth });
});

Hent nuværende dag

Her bruges et array af dagsnavne og getDay() til at finde den aktuelle dag.

Endpoint: /days/v1

const days = ["Sunday", "Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday"];
app.get("/days/v1", (req, res) => {
  const currentDay = days[new Date().getDay()];
  res.send({ data: currentDay });
});
