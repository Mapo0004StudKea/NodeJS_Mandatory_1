Can create a server with Express from memory.


Opsætningsguide til at oprette en server med Express i Node.js


Hvad er Express?

Express er et webframework til Node.js, som gør det nemt at oprette webservere.


Opsætning af en Express-server

Opret et nyt Node.js-projekt

Start med at initialisere et projekt og installere Express:


  mkdir testprojekt && cd testprojekt
  npm init -y # initialisere projektet
  npm install express

lavt en simpel Express-server

Lav en ny fil kaldet app.js og indsæt følgende kode:

  
    const express = require('express');

    const app = express();
    
    app.use(express.json());
    
    // Definér en simpel route
    app.get('/', (req, res) => {
        res.send('Velkommen til min Express-server!');
    });
    
    // Start serveren på port 8080
    const PORT = 8080;
    app.listen(PORT, (error) => {
        if (error) {
            console.log("Error starting the server", error);
            return;
        }
        console.log("Server is running on port", PORT);
    });

Kør serveren

Kør følgende kommando i terminalen:

  
    node app.js

Test serveren

Åbn browseren og gå til http://localhost:8080 for at se Express-server.
