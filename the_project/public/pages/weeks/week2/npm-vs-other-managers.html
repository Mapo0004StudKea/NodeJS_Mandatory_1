Node Package Manager

NPM (Node Package Manager) er standardpakkehåndteringen til Node.js. Den bruges bland andet til at:

  • Installere og administrere tredjepartsbiblioteker (f.eks. Express)
  • Automatisere scripts (f.eks. køre tests eller starte serveren).

NPM-kommandoer:

Kommando Beskrivelse
  • npm init
  • npm install (pakke)
  • npm install -g (pakke)
  • npm install
  • npm run (script)
  • npm uninstall (pakke)

  • Installerer en pakke lokalt
  • Installerer en pakke globalt
  • Installerer alle afhængigheder fra package.json

    Eksempel:

    
  npm init -y             # Opretter package.json med standardværdier
  npm install express     # Installerer Express lokalt
