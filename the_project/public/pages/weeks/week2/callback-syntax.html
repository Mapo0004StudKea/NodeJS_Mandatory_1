Can write callback functions with different syntax.


Callback med en Navngiven Funktion

Man kan definere en funktion og bruge den som en callback ved at give dens navn som argument.


  function greet(name) {
    return `Hej, ${name}!`;
  }
  
  function genericPerformer(name, action) {
      return action(name); // Kalder callback-funktionen
  }
  
  console.log(genericPerformer('Martin', greet));
  // Output: "Hej, Martin!"

Callback med en Anonym Funktion

Man kan definere callback'en direkte som en anonym funktion (uden navn) inde i funktionskaldet.


  console.log(genericPerformer('Martin', function(name) {
    return `${name} elsker at sove`;
  }));
  // Output: "Martin elsker at sove"

Callback med en Arrow Function

En arrow function er en kortere måde at skrive en anonym funktion på.


  const codingAction = (name) => `${name} elsker at kode`;
  console.log(genericPerformer('Martin', codingAction));

Callback med en Inline Funktion

Man kan også skrive callback'en direkte inde i funktionskaldet som en one-liner.


  console.log(genericPerformer('Mia', (name) => `${name} elsker at kode`));
  // Output: "Mia elsker at kode"
