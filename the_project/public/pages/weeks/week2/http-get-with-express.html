Understands how to send data with GET requests over HTTP and knows how to do it in Express.


GET-requests med Express

Hvad er en GET-request?

En GET-request bruges til at hente data fra en server. Når man besøger en hjemmeside, sender ens browser typisk en GET-request til serveren for at hente indholdet af siden.

Sådan kan man sende en GET-request i Express

Her er et eksempel på, hvordan man opsætter en Express-server, der håndterer en GET-request:

Express Server Guide

Opret en ny GET-route

    
    // opetter en ny GET-route
    app.get('/Hej', (req, res) => {
        res.send('Hej, velkommen til min server!');
    });

Sådan tester man GET-requesten:

  1. Åbn en terminal og kør serveren med node app.js.
  2. Åbn en browser og gå til http://localhost:8080/hej.
  3. Bør se beskeden: "Hej, velkommen til min server!".

Forståelse af GET-requests i Express

Query Parameters

Man kan sende data via URL'en ved hjælp af query parameters.

    
    app.get("/search", (req, res) => {
      res.send({ data: `Du søgte efter: ${req.query.q}` });
    });

Eksempel: Hvis man går til http://localhost:8080/search?q=kat, får man svaret:

{ "data": "Du søgte efter: kat" }

Route Parameters

Ruteparametre giver mulighed for at sende data direkte i URL-strukturen.

    
    app.get("/yourfavoritenumber/:favoriteNumber", (req, res) => {
      res.send({ data: `Dit yndlingstal er: ${req.params.favoriteNumber}` });
    });

Hvis man går til http://localhost:8080/yourfavoritenumber/7, får man svaret:

{ "data": "Dit yndlingstal er: 7" }

Redirect

Man kan omdirigere brugeren til en anden route:

    
    app.get("/redirecttofavoritethings", (req, res) => {
      res.redirect("/favoritethings/tulips/sealions");
    });

Hvis man går til /redirecttofavoritethings, bliver man sendt videre til en route, der viser yndlingsblomst og -dyr.

Proxy Requests

Man kan hente data fra en ekstern hjemmeside og sende dem til klienten:

    
    app.get("/proxy", (req, res) => {
      fetch("https://www.google.com")
          .then(response => response.text())
          .then(result => res.send(result));
    });

Gå til /proxy for at se Google's hjemmeside returneret gennem Express-serveren.
