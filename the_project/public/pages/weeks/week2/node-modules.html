Understands node_modules.


Hvad er node_modules?

node_modules mappen er hvor Node.js gemmer alle eksterne pakker (moduler), som installeres via npm (Node Package Manager).

Hvordan oprettes node_modules?

Når man kører kommandoen:

npm install

Installeres alle afhængigheder defineret i package.json-filen, og de gemmes i node_modules mappen.

Eksempel på package.json:

package-json

Hvordan bruges node_modules i Express?

Når express installereres, kan man importere det i sin applikation:

    
      const express = require('express');
      const app = express();

Hvorfor skal node_modules ikke pushes til Git?

node_modules mappen er ofte stor og behøver ikke at være i ens versionskontrolsystem. I stedet kan det genskabes ved at køre npm install baseret på package.json.

Undgå at tilføje den til Git ved at bruge en .gitignore-fil.
