Understands the different purposes of package.json and can set it up by hand.


Hvad er package.json?

package.json er em configurationsfil some bruges i Node.js projekter til at:

  • Gemme metadata om et projekt som f.eks. navn, version m.m.
  • Definere dependencies som er biblioteker som projektet skal bruge.
  • Includere scripts for at automatisere opgaver som f.eks. start server

Hvordan kan man lave en package.json fil

Man kan selv lave en package.json fil ved at bruge:

npm init.

Forklaring for hvad nogle af Key Fields er:

  • name: Er projektets navn
  • version: Er projektets version.
  • description: Er en kort forklaring for hvad projektet gør.
  • main: Entry point fil (Vi bruger ofte app.js).
  • scripts: Custom commandoer (npm start).
  • dependencies: Externe pakker som projektet skal bruge.
  • license: Definere the projektets license type.
