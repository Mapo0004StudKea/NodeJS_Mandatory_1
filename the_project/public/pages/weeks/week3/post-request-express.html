Sådan laves en POST-request i Express

Hvad er en POST-request?

En POST-request bruges til at sende data til en server for at oprette en ressource. I Express kan det håndteres ved anmodninger som app.post().

Eksempel på en POST-request i Express

Her er et eksempel fra klassen, der håndterer en POST-request til /favoritepoliticians:


  app.use(express.json()); // Tillader parsing af JSON i POST-body

  app.post("/favoritepoliticians", (req, res) => {
    console.log(req.body); // Logger den modtagne data
    res.send({ data: req.body }); // Sender dataen tilbage som respons
  });
