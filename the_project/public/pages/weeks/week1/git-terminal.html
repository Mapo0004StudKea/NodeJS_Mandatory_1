Can work with Git in the terminal


Har i Full Stack Node.js semesteret anvendt Visual Studio Code

For at oprette et projekt kan følgende kommandoer anvendes:

  • mkdir my-project
  • cd my-project
  • git init
    •

    Git init bliver brugt til at initialiserer et nyt repository


    Tilføj filer til git

    når der er oprettet eller redigeret i filer kan man tilføje dem til git:

    git add .

    Kan også gøres med en specifik fil

    git add filename.txt

    Gem ændringer med en committ

    git commit -m "My first commit"

    Opret et repository på GitHub

    1. Gå til GitHub og log in
    2. Klik på New Repository
    3. Giv det et navn
    4. ælg "Public" eller "Private".
    5. Klik på Create repository.

    Forbind den lokale mappe til GitHub

    Når man har oprettet sit repository på GitHub, får man en url som kan bruges til at forbinde sin mappe til GitHub.

  • git remote add origin https://github.com/username/projectname
  • git push -u origin main
    •

    Med disse kommandoer kan ens projekt blive uploadet til GitHub


    Hent ændringer fra GitHub

    git pull origin main

    Upload (push) ændringer til GitHub

  • git add .
  • git commit -m "updated code"
  • git push origin main
