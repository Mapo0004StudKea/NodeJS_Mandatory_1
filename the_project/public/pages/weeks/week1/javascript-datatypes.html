Understand and work with data types in Javascript


JavaScript understøtter flere datatyper, som kunne være:

    Tal

    Strænge

    Booleans

    Objekter

    BigInt

    symbol

    null

    undefined

Hvor Tal repræsenterer numeriske værdier, strenge repræsenterer tekst, booleans repræsenterer sande eller falske værdier, objekter gemmer nøgleværdi-par, null repræsenterer det tilsigtede fravær af enhver objektværdi, og udefineret repræsenterer fraværet af en defineret værdi.

Datatyper i JavaScript definerer den type data, en variabel kan indeholde.
JavaScript indeholder primitive datatyper, de er simple og indeholder kun én værdi.
og ikke-primitive datatyper (objekter) er mere komplekse og kan indeholde flere værdier.


primitive datatyper

    Streng

    En streng-datatype er en gruppe af tegn eller tekstindhold omgivet af enkelte anførselstegn (' ') eller dobbelte anførselstegn (" ")

    Ekempel kunne være:

    var string = 'Hello, World!'; // Output: Hello, World!
    Tal

    En tal-datatype er en numerisk værdi. Det kan være et heltal (positivt eller negativt), floating-point eller eksponentielt tal.

    Ekempel kunne være:

  • var a = 47; // positiv int værdi
  • var b = 629.5; // floating point
  • var c = -5; // tal med negativ værdi
    •
    Boolean

    Boolean er en logisk datatype, der bruges til at sammenligne to variable eller kontrollere en betingelse. Den har kun to værdier, sand eller falsk.

    Ekempel kunne være:

  • var a =10;
  • var b=9;
  • a==b // returnere falsk
    •
    symbol

    Symboldatatypen i JavaScript definerer en egenskab for et objekt, som er privat for objektet. Værdien med symboldatatypen kan omtales som en symbolværdi. Symbolet refererer til 'nøglen' for nøgleværdi-parret af et objekt. Hvert symbol er unikt. To symboler med samme nøgleværdier er ikke ens.

    Ekempel kunne være:

  • let value1 = Symbol("value");
  • let value2 = Symbol("value");
  • console.log(value1 == value2); // falsk
    •
    null

    En nulværdi betyder ingen værdi. Det betyder en tom værdi.

    Ekempel kunne være:

  • var x = null;
  • console.log(x); // returnere null
    •
    Undefined

    En udefineret datatype betyder, at en variabel er deklareret og ikke tildelt nogen værdi. Det er standardværdien for en variabel, der ikke er defineret.

    Ekempel kunne være:

  • var x; // undefined
  • console.log(x); // returnere undefined
    •

ikke-primitive datatyper

    Objekter

    En objektdatatype gemmer flere værdier eller dataindsamling vedrørende egenskaber og metoder. Den indeholder nøgleværdi-par i sin adresse. Objektets egenskaber skrives som nøgle: værdipar. Hvert par er adskilt af kommaer, omgivet af krøllede klammeparenteser {}. Nøglen skal være en streng, og værdien kan være af enhver datatype.

    Ekempel kunne være:

  • var student = {firstName: 'martin', class: 'Kea_DAT_Node.js_2025_Spring' };
