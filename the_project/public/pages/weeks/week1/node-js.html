Can explain what Node.js is and how to run files with it

Node.js er et open-source og cross-platform JavaScript runtime environment . Dette betyder at man kan bruge javascript til både hjemmesiden og serveren, der styrer hjemmesiden.


Node.js er hurtigt og effektivt, da det kan håndtere mange ting på en gang uden at blive langsommere. Det bruges ofte til at bygge webapplikationer, API'er og værktøjer, der kører i kommandolinjen.


Når man har installeret Node.js, kan man køre et Node.js-program ved at bruge kommandoen node i terminalen og angive navnet på den fil, man vil køre.


Hvis hovedfilen for Node.js-applikationen hedder app.js, kan man køre den ved at skrive:

node app.js

Ulemper

Der er dog nogle ulemper ved kun at køre node app.js i det man skal genstarte hver gang ændringer bliver lavet.
det kan også blive tidskrævende hvis projektet er større og kan blive irriterende at skulle genstarte efter hver ændring.
