Can import/export in both ways in Node.js (CommonJS and ES Modules). Understands the pros of using ES Modules.


Import/Eksport i Node.js

Node.js og javascript giver to hovedtyper af import/eksport: CommonJS (require/module.exports) og ES Moduler (import/export). I HTML vil der primært blive brugt ES Moduler.

CommonJS (require/module.exports)

CommonJS bliver primært brugt i Node.js-miljøer, bruger require og module.exports til at definere dependencies og moduler. Er ikke understøttet i browsere.

Skal have "type": "commonjs" i package.json filen for at kunne blive brugt

Eksempel på eksport

Bliver exporteret med module.exports


  // module.js
  function subtract(a, b) {
    return a - b;
  }

  module.exports = {
    subtract: subtract
  };

Eksempel på import

Bliver importeret med require('...')

      
    // app.js
    const { subtract } = require('./module.js');
  
    console.log(subtract(10, 4)); // Output: 6

ES Moduler (import/export)

ES Moduler er den mere moderne måde at importere og exportere i node.js, og understøttes i de fleste moderne browsere.

Skal have "type": "module" i package.json filen for at kunne blive brugt

Eksport

Man kan eksportere funktioner, variabler eller klasser fra en JavaScript-fil.

// module.js
  export function add(a, b) {
    return a + b;
  }

  export const myVariable = 42;

Import

Man kan importere eksporterede værdier i en anden JavaScript-fil.

// app.js
  import { add, myVariable } from './module.js';

  console.log(add(5, 3)); // Output: 8
  console.log(myVariable); // Output: 42

Brug i HTML

For at bruge ES Moduler i HTML skal type="module" tilføjes til <script>-tagget.

<script type="module" src="app.js"></script>
