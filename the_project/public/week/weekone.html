    Can work with Git in the terminal

    Using the terminal in Visual Studio Code

  • mkdir my-project
  • cd my-project
  • git init
    •

    a git repository have now been created in the folder

    Add files to git

    when created or edited a file add to git with:

    git add .

    or a specific file

    git add filename.txt

    save changes with a commit

    git commit -m "My first commit"

    Create a repository in GitHub

    1. Go to GitHub and log in
    2. Click new repository
    3. add a name
    4. Choose "public" or "private
    5. Click create repository

    Connect the repository to the local git repository

  • git remote add origin https://github.com/username/projectname
  • git push -u origin main
    •

    get changes from main branch

    git pull origin main

    upload (push) changes to GitHub

  • git add .
  • git commit -m "updated code"
  • git push origin main
